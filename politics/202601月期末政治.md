## 第16题（8分）

**题目：** 结合材料，运用《经济与社会》知识，国家应如何整治"内卷式"竞争。

**你的答案：**
1. 推动有效市场和有为政府相结合，制定统一市场建设条例，发挥市场活力，促进经济高质量发展。
2. 政府要智能高效，精准调控，推动企业创新，强化知识产权保护，优化资源配置。
3. 建设高标准市场体系、完善公平竞争制度，维护统一开放，竞争有序的市场体系。

**批改：**

| 要点 | 评价 |
|------|------|
| 有效市场与有为政府相结合 | ✅ 正确，且与材料高度契合 |
| 制定统一市场建设条例 | ✅ 直接来源于材料，体现制度层面 |
| 政府精准调控 | ⚠️ 表述过于笼统，"智能高效"不是规范术语 |
| 强化知识产权保护 | ✅ 有一定道理，但未充分展开 |
| 完善公平竞争制度 | ✅ 符合教材表述 |

**主要失分点：**
- 缺少**规范市场秩序、反对垄断和不正当竞争**的角度（反行政性垄断、市场分割是材料核心）
- 缺少**发挥政府监管职能**的具体表述（材料提到制度、监管、引导三层面）
- 答案较为零散，层次感不强，逻辑未完全围绕材料展开
- "知识产权保护"属于《经济与社会》外延，要注意知识范围限定

**参考得分：约4～5分/8分**

**参考要点：**
> ①充分发挥市场在资源配置中的决定性作用，完善价格、竞争、退出机制，破除行政性垄断和市场分割；②更好发挥政府作用，加强市场监管，规范市场秩序，依法查处不正当竞争行为；③完善公平竞争制度，建设全国统一大市场，营造公平、开放、透明的市场环境；④引导企业转向高质量、差异化竞争，推动经济高质量发展。

## 第17题（8分）

**题目：** 运用《政治与法治》知识，说明中国特色解决民族问题的道路是一条怎样的道路。

**你的答案：**
中国特色主义道路是一条以中华民族根本利益和整体利益为出发点，以各民族共同团结奋斗为目标，以人民当家作主、各民族平等、实行民族区域自治为制度的中国特色道路。

**批改：**

| 要点 | 评价 |
|------|------|
| 以中华民族根本利益为出发点 | ✅ 来源于材料 |
| 各民族共同团结奋斗 | ✅ 正确 |
| 人民当家作主、各民族平等 | ✅ 有体现 |
| 民族区域自治 | ✅ 提到但未展开 |

**主要失分点：**
- 答案结构**几乎只是一句话**，未分点展开，不符合8分题的作答要求
- 缺少**坚持马克思主义民族理论与中国具体实际相结合**的角度（材料第一句即点明）
- 缺少**坚持"两个结合"（马克思主义+中华优秀传统文化）** 这一重要表述
- 缺少**维护国家统一与实行民族区域自治相统一**的具体阐述
- "民族区域自制度"有错别字（应为"自治"）

**参考得分：约3分/8分**

**参考要点：**

## 第18题（9分）

**题目：** 运用经济全球化与中国的知识，分析中国—东盟经贸合作对我国推进高水平对外开放的意义。

**你的答案：**
1. 减少关税壁垒、优化贸易政策，有利于降低区域内贸易成本，促进各国经济发展，推动区域经济一体化。
2. 坚持创新发展新质生产力，加强中国—东盟产业合作，提升双方经济合作深度。
3. 顺应经济全球化一体趋势，促进商品流动，提高资源利用效率。

**批改：**

| 要点 | 评价 |
|------|------|
| 降低贸易成本、推动区域一体化 | ✅ 方向正确 |
| 新质生产力、产业合作 | ⚠️ 与题目"对外开放意义"关联不够紧密 |
| 顺应全球化趋势 | ⚠️ 过于宽泛，未结合材料具体内容 |

**主要失分点：**
- 题目问的是对**我国推进高水平对外开放**的意义，角度应聚焦**我国**，而非泛泛谈双方
- 缺少**拓展对外开放广度和深度**、**构建开放型经济新体制**等教材核心表述
- 缺少**维护多边贸易体制、推动构建人类命运共同体**的角度
- 缺少对材料信息的有效利用（新兴领域、绿色经济、数字经济、行动计划等）
- 答案9分题只答了3点，内容单薄

**参考得分：约4分/9分**

**参考要点：**
> ①有助于拓展对外开放的广度和深度，推动贸易和投资自由化便利化，构建开放型经济新体制；②深化在新兴领域、绿色经济、数字经济等领域合作，推动我国产业升级和高质量发展；③有助于维护以规则为基础的多边贸易体制，推动构建新型国际关系和人类命运共同体；④为我国企业"走出去"创造更多机遇，提升我国在区域经济合作中的影响力。

## 第19题（12分）

**题目：** 说明调解协议约定内容的依据以及此纠纷被成功化解的意义。

**你的答案：**
1. 张某经常免费搭乘李某的私家车，属于好意同乘。但李某在行驶过程中边接电话边开车，并且超速，对事故负重大责任。
2. 张某父母同意减少赔偿数额，符合民法典的原则。

意义：
1. 促进邻里和睦与工作关系的和谐。
2. 通过调解解决纠纷，提高纠纷解决效率。

**批改：**

| 要点 | 评价 |
|------|------|
| 好意同乘认定 | ✅ 正确 |
| 重大过失认定 | ✅ 正确，但未点明法律后果（即不适用减轻条款） |
| 协议依据 | ❌ 仅说"符合民法典原则"，未说明**双方协商一致、意思自治**原则 |
| 意义1 | ⚠️ 过于简单，未结合法律术语 |
| 意义2 | ✅ 基本正确，但未展开 |

**主要失分点：**
- **依据部分**核心缺失：
  - 好意同乘规则本应减轻李父责任，但因其有**重大过失**，依法不适用减轻条款，故李父应承担主要责任——这个法律推导链条不完整
  - 协议内容体现了**当事人意思自治原则**和**民事主体平等协商**原则，未提及
- **意义部分**严重不足（12分题仅两点，且内容极为简单）：
  - 缺少**化解社会矛盾、维护社会稳定**
  - 缺少**节约司法资源，降低诉讼成本**
  - 缺少**有利于当事人修复关系**
  - 缺少**弘扬社会主义法治精神，引导公民依法维权**等角度

**参考得分：约4～5分/12分**

**参考要点（依据）：**
> ①根据民法典好意同乘规则，李父作为非营运车辆驾驶人，原本可减轻赔偿责任；②但李父边驾车边接听手机、超速50%，构成重大过失，依法不适用减轻条款，应承担全部赔偿责任；③双方在法官调解下自愿协商，体现了民法典的平等原则和意思自治原则，协议内容合法有效。

**参考要点（意义）：**
> ①及时化解民事纠纷，维护当事人合法权益；②发挥调解的多元化纠纷解决机制作用，节约司法资源；③有助于修复当事人之间的社会关系，促进社会和谐稳定；④引导公民理性依法维权，弘扬法治精神。

## 综合建议

**1. 审题要精准。** 每道题都有明确的知识范围限定（《经济与社会》《政治与法治》《法律与生活》），答题时要严格对应教材知识点，不能跨模块乱用。

**2. 材料要"吃透"。** 你的答案普遍存在"脱离材料"的问题。主观题的关键在于**材料信息+教材原理**的有机结合，每个要点最好能做到"材料中有依据，教材中有支撑"。

**3. 答题要分层，字数要够。** 8分题至少4个要点，12分题至少4～6个要点，每点要有完整表述（原理+联系材料），一句话一个点是拿不到高分的。

**4. 规范用语。** 避免口语化表达，使用教材中的规范术语，如"意思自治""市场在资源配置中起决定性作用""完善公平竞争制度"等。

**5. 法律题要注意推理链条。** 第19题这类题型要求做到：**认定事实→适用法律条文→得出结论**，三步缺一不可。
